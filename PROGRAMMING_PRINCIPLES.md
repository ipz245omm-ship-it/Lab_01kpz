# Принципи програмування

## 1. Принцип KISS

У проєкті використовуються прості та зрозумілі конструкції JavaScript. Для пошуку HTML-елементів застосовується `querySelectorAll()`, а для обробки — `forEach()`. Це робить код простим для читання та підтримки.

**Приклад з коду:**
- index1.html — робота з абзацами та таблицею через `querySelectorAll` і `forEach`.

---

## 2. Принцип DRY

У роботі з таблицею використовується вкладений цикл, що дозволяє не дублювати код для кожної комірки окремо.

**Приклад з коду:**
- index1.html — цикл `forEach()` для рядків і комірок таблиці.

---

## 3. Захисне програмування

Перед використанням елементів DOM виконується перевірка їх існування, щоб уникнути помилок.

**Приклади з коду:**
- `if (paragraphs[0])`
- `if (paragraphs[1])`
- `if (paragraphs[2])`
- `if (!txt) return txt;`

---

## 4. Читабельність коду

Код структурований і поділений на логічні блоки з коментарями, що спрощує його розуміння.

**Приклад:**
- index1.html — окремі частини для абзаців, таблиці та роботи з рядками.
