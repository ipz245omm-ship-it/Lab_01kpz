# Принципи програмування

## 1. Принцип KISS

У проєкті використовуються прості та зрозумілі конструкції JavaScript. Для пошуку HTML-елементів застосовується метод `querySelectorAll()`, а для їх обробки — цикл `forEach()`. Завдяки цьому код легко читати, розуміти та підтримувати.

**Приклади:**
- `index1.html` — пошук усіх абзаців за допомогою `querySelectorAll('p')`.
- `index1.html` — перебір рядків і комірок таблиці за допомогою `forEach()`.

---

## 2. Принцип DRY

Під час роботи з таблицею не дублюється код для кожної комірки окремо. Замість цього використовується вкладений цикл, який автоматично обробляє всі рядки та комірки.

**Приклад:**
- `index1.html` — використання вкладених циклів `forEach()` для зафарбовування комірок таблиці.

---

## 3. Захисне програмування

Перед зверненням до елементів виконується перевірка їх існування. Це дозволяє уникнути помилок під час виконання програми, якщо потрібний елемент відсутній.

**Приклади:**
- `if (paragraphs[0])`
- `if (paragraphs[1])`
- `if (paragraphs[2])`
- `if (!txt) return txt;`

---

## 4. Читабельність коду

Код структурований та містить пояснювальні коментарі. Програма поділена на логічні частини відповідно до завдань лабораторної роботи, що спрощує її розуміння та подальшу підтримку.

**Приклади:**
- Частина 1.1 — робота з абзацами.
- Частина 1.2 — робота з таблицею.
- Завдання 2 — робота з рядками.
